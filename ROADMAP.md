# Roadmap

- Publish a clean local-first open-source package.
- Provide fictional examples and tests.
- Support B-roll markdown plus SRT plan generation.

- Improve image material templates for multiple editor versions.

## v0.3

- Add optional OpenTimelineIO / XML export.
- Add release automation.
- Add broader documentation for creators and maintainers.
