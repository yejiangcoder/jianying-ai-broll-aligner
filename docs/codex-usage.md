# Codex Usage

This project was developed with Codex to reduce repetitive editing work in AI-assisted video production workflows.

Codex is useful in this project for:

- local Python tooling;
- draft file parsing;
- subtitle matching;
- test generation;
- documentation;
- release workflows;
- issue triage and maintainer automation.

## Maintainer Workflow

Typical Codex-assisted maintenance tasks:

1. Add a new fixture for a draft structure.
2. Update the adapter or parser.
3. Generate focused regression tests.
4. Run tests locally.
5. Update documentation and examples.
6. Prepare a small release.

## Why It Matters

Video editor draft formats can change between versions. Codex helps maintainers inspect fixtures, isolate behavior changes, and update the local tooling without turning each editor update into manual trial-and-error work.
